# 📦 Product CRUD App

A products management app with full **Create, Read, Update and Delete** operations and search, built with vanilla JavaScript and persisted in the browser using localStorage.

## 📖 Overview

A classic CRUD interface for managing products (name, price, category, description). All data is stored in the browser's **localStorage**, so your list persists across page reloads without any backend.

## ✨ Features

- ➕ **Create** — add new products through a validated form
- 📖 **Read** — products rendered in a dynamic table
- ✏️ **Update** — edit any existing product
- 🗑️ **Delete** — remove a single product or clear all
- 🔎 **Search** — filter products as you type
- 💾 Data persistence with localStorage

## 🛠️ Tech Stack

- HTML5 & CSS3
- Bootstrap
- Vanilla JavaScript (ES6+, localStorage)

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/crud-project.git
cd crud-project
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).
